# 2026年度 写真展作品 Web掲載同意確認（ひな形）

この文書は部内運用用のひな形です。展示ごとの会期、サイトURL、公開期間、問い合わせ先を確定してから使用してください。

## 1. 掲載の目的

法政大学 小金井写真部は、次の目的で出展作品のWeb掲載を希望します。

- 写真展来場者による作品閲覧
- お気に入り作品アンケートの実施
- 写真展の案内および活動記録

Web掲載への同意は任意です。同意しない場合も、写真展への出展その他の部活動上の取扱いで不利益はありません。

## 3. 作者による確認

各項目を確認してください。

- [ ] 私は、この作品の著作者であり、掲載を許可できる権限を有しています。
- [ ] Webサイトが一般公開され、閲覧者が画像を保存・撮影等する可能性があることを理解しました。
- [ ] Web表示のための縮小、圧縮、形式変換、メタデータ削除を許可します。
- [ ] 上記の作者名、作品名、説明文の掲載内容を確認しました。
- [ ] 被写体の肖像、第三者の著作物、撮影場所の規約等、掲載に必要な確認を行いました。
- [ ] 掲載取下げを希望する場合の連絡先を確認しました。

## 4. 利用範囲の選択

同意する項目だけにチェックしてください。

- [ ] 写真展サイトの作品一覧への掲載
- [ ] 写真展サイトのアンケート画面への掲載
- [ ] 会期終了後の一般公開アーカイブ
- [ ] 部員だけが閲覧できる認証付きアーカイブ
- [ ] 写真部公式SNSでの告知・結果発表への掲載
- [ ] 写真展のDM、ポスター等の広報物への掲載

チェックのない用途には使用しません。新しい用途で使用する場合は、改めて作者へ確認します。

## 7. 運用担当者向け注意

- 口頭だけで終わらせず、後から作品・作者・同意範囲を確認できる形で保存する

- 会期後アーカイブは一般公開と部員限定を分けて確認する
- 同意撤回や訂正の連絡を受けた日時と対応内容を記録する
